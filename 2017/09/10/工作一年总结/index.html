工作一年总结

2017-09-10

  2. 校园与社会

  4. 2017年2月 新的开始

古人云，学而不思则罔，思而不学则殆。边学习，边思考才能够真正参悟学习的课本中的真谛。生活又何尝不是，在忙碌的生活中需要有那么一段时间，或者是那么一个时间点去回顾之前一段的时间。回忆那一段时间里面自己成长了多少，得到了什么，又失去了什么。亦或是与自己之前的期望是否一致，是达到了当初的目标还是没有达到当初的目标。而今天就恰好是那么一个时间点，说实在的，其实已经拖延了一个月的时间。栽一棵树最好的时机是在十年前，其次是现在。

2016年7月 步入社会

作为一个大专毕业的学生，社会起步就比本科毕业的学生要低一些。虽然又花费了两年的时间去上了成人教育的本科，但终归不是正规军。刚刚毕业的时候心里就只有一个念头，那就是找到一份工作，工资什么完全没有要求。如同一只流浪远方的打工仔，有一个窝就已经很满足了。那个时候也曾经后悔过，后悔花费了两年的时间去上了一个意义不大的成人本科。但现在想一想，在哪个时间点去上成人本科或许是我最好的选择。

初步步入社会我就像一张白纸一样，除了在大专还没有毕业就被班主任推荐到北京实习的那三个月时间，其他的一无所有。大学中浪费时间的后遗症立马显现出来，该会的不会，会的又只是皮毛。从学校毕业后回到家，我就立马开启了我的找工作之旅。我不知道为什么，那个时间心里压力特别大，仿佛错过了那几天就永远找不到工作似的。

还好后来凭借大学里两年的国家励志奖学金、蓝桥杯省一等奖、无锡市三号学生和低廉的价格（我是这么认为的），一家“企业”终于是收留了我，我第一个月的工资是 2000 元。

校园与社会

都说校园和社会完全不一样的世界，刚刚上大学那会就听老一辈的人，或者是互联网上的网友说过这一句话。当自己快要毕业的时候也不断的提醒着自己，自己也仿佛是做好了步入了社会的准备，也天真的以为自己已经准备好了。但真正进入的社会的时候，与其说自己是没有准备好，倒不如说是完全没有什么准备。

在校园里面不算最优秀的，但也是很优秀的。作为班级班长的我，年年班级第一，拿了两年的国家励志奖学金，而且还是第一批就入党的人，到公司上班的第一周我就蒙圈了，仿佛是第一次进入大观园的刘姥姥。上班前是怕自己找不到工作，上班后是怕自己能力不够，而被炒鱿鱼。

7 月底的南京正是雨季的时候，暴雨是我现在对那一段的唯一记忆。只记得面试的那一天下着暴雨，打伞已经没有用了，我踩着水坑，冒着大雨走进了第一家公司所在的大楼。在厕所里面倒了倒鞋子里面的水，擦了擦脸上的水，就这样踏上了程序员的征途。

没有压力的半年时间

说实在的，我的第一份工作并不是很理想。工作了 3 个月之后我才算是真的成为公司的员工。因为我所在的这家公司 10 月份才拿到营业执照，而我很荣幸的成为公司的第一批员工。作为所在公司唯一的一个程序员，向同一个办公室另外一家公司的项目经理模仿和学习，就这样度过了 6 个月的时间。拿着项目经理给的框架，摸爬滚打的完成了我人生的第一个项目。（后来在我之前的前端同事口中得知那个项目被后来招的项目经理完全弃用了，公司用 Python 去实现这个项目了）

六个月的时间说长不长，说短不短。这一段时间时走时停，也不知道外面的世界是什么样子，也不知道自己要成为什么样的人。塞翁失马，焉知祸福，空闲的工作时间给我了大量自己学习的时间；公司人少给了我机会接触到服务器和微信支付。我感谢我的第一家公司，也感谢当初的我，感谢第一家公司给了我步入社会的缓冲器，感谢当初的我没有忘记为什么而出发。

2017年2月 新的开始

时间来到了 2017 年 2 月，根据原定计划我离开了我的第一家公司，来到了现在的公司。这一次面试比上一次面试底气充足了许多，也更有自信了。但是也暴露我很多的不足。第一家公司用的是小众框架 JFinal,虽然我花了很多时间去补 Spring、SpringMvc 等框架的知识，但是自己并没有实践过；第二就是 Oracle 数据库，一直都没有使用过；最后就是没有团队工作的经验。就这样勉勉强强的通过了面试，开始了我新的旅途。

充满挑战的半年时间

这次找的是一家有 200 多人的公司，开发人员大约有 100 人左右，第一次感受到了公司的氛围。湖光猎影般的总结今年上年半年在公司的工作就是改代码、写代码、设计代码、分析需求。我觉得自己很幸运，也觉得自己还需要更多的磨砺。这半年的时间也是走走停停，有的时候效率很高，有的时候就完全是停滞不前。我需要磨砺自己的意志，提高自己写代码和分析代码的水平，增长工作外学习的时间。

工作第二年的计划

如今的我已经收拾行囊开始我工作第二年的征途，在前不久面试一个新人的时候真的感觉到现实的残酷。虽然来面试的人有一年多的工作经验，但是我和我的同事都不感觉他符合一年工作经验的程序员的要求。这不经让我背后一凉，我真的就符合一年工作经验的要求吗？

我庆幸我才工作一年，还有时间去学习，去弥补过往一年的不足；也很悔恨，后悔过往一年中浪费的时间。但无论怎么样，过去都已经过去，好的坏的，都让他随风而散。就以现在作为起点，一年后的我作为目标。奋力过好这一年的每一天，每一秒。

目标：
0.学习英语
1.高质量的看完《Java 编程思想》和《代码大全》
2.系统的学习 Oracle 和 MySql 数据库
3.学习 Spring 框架源代码
4.写出一些高质量的博客
5.每天读书
6.将体重缩减至 75 kg

额外目标：
0.看懂 《Thinking in Java》
1.学习 JVM
2.学习 前端知识
