IDEA 打开多个项目

2018-03-29

简单的说只需要以下几步：
1.将需要同时打开的模块放在一个文件夹下，e.g. AModel 和 BModel 同时放在 Project 文件夹下。
2.在 IDEA 初始导入项目页面选择 open -> 打开 Projcet 文件夹
3.如果是 Maven 项目点击界面右侧的 Maven Projcets 打开 Maven Projcets 管理界面，点击上面的 Add Maven Projects，弹窗选择 AModel 和 BModel 项目下的 pom.xml 即可
