Nginx日志统计接口被调用次数

最近公司系统总是出问题，看到项目经理通过日志统计出了后台 API 接口的调用次数（统计在了 Excel 表中）。因为之前自己都没有做过类似的统计，今天看到了不禁在想他是怎么统计的，自己能否也可以做出这样的报表。

初步设想他应该是统计 Nginx 的 access.log 里面所有的请求，然后对数据进行过滤和分组统计次数。
