软路由——1.0介绍

2020-01-02

文章目录

  1. 为什么要使用软路由
  2. 问答环节

如果你不知道什么是 SS，什么是梅林，或者对软路由一点都不了解，请先了解。

生命不息，折腾不止。2019年末开始折腾软路由这个东西，什么是软路由呢，可以简单的理解为由一个小型电脑，靠着某种软件设置，达成路由器功能。一般这种小型电脑都是使用非常低功耗的CPU组成，不是我们家庭常用的那种，体积也很小。

软路由可以干什么呢？它基本上是为 200M 以上宽带科学上网使用的。划重点：200M，科学上网。
家庭宽带决定了它并不会有很多设备同时上网，所以买一个靠谱的路由器基本就足够了。我估计 100 块左右的 TP-link 和 700块左右的 ASUS-AC86U，在 50M 的宽带下很多时候的上网体验可能是差不多的，并不会有太大的区别。

做程序员的估计有很多人都有科学上网的需求，毕竟现在网络环境越来越不好。比如说 GitHub，我记得前几年都是可以正常访问的，但是最近开始越来越慢了。不开科学上网访问基本慢如狗，拉个库就像百度云下载一样，只有几十 KB。

我知道的科学上网的方式就那么几种————VPN,hosts,SS,SSR,V2ray 等。当然这些都需要配合着软件使用，我自己用的是SSR，购买的是一个机场的服务。这里我就不介绍什么是 SS啊，SSR了，如果没有用过，请自行 Google。我的 Mac 设备使用的是 Surge 或者 ShadowsocksX-NG-R 软件使用，Windows 电脑使用的是 ShadowsocksR，IOS 设备使用的是 Shadowrocket。如果你没有用过类似的服务，估计已经晕头转向了，不同的设备需要使用不同的软件配合 SSR 服务使用。

为什么要使用软路由

我在使用科学上网的时候就遇到了以下几个问题：
1.不同设备需要使用不同的软件，有学习成本。
2.并不能跑满宽带，速度差强人意。
3.无法分享，有的时候一个新设备没有软件，必须要安装并配置后才能使用。

针对以上几点问题，软路由可以很好的解决。软路由有很多种使用的方式，我这里刚刚学习折腾，所以并没有把它作为家里所有网络的中枢，而是将它作为一个旁路由来使用。什么是旁路由呢？简单来说就是一个网络中间交换设备。

原本我们上网是这样的:
电脑/手机等设备 -> 路由器 -> 互联网

加入了旁路由我们上网是这样的：
电脑/手机等设备 -> 旁路由 -> 路由器 -> 互联网

这样有什么用呢？
我们可以将科学上网软件安装在旁路由上，这样我们各种各样的上网设备就不用再安装各种各样的科学上网软件，也只需要配置一遍就可以做到科学上网了，并且由于其系统的特殊性，基本上一个好一点的 CPU 完全可以跑满带宽。

问答环节

Q：为什么不用路由器？梅林不是可以吗？
A：支持梅林系统的路由器一般都是比较高端的路由器，很多路由器并不支持，并且就算支持由于路由器的 CPU 是 ARM 架构，性能羸弱，并不适合使用科学上网软件。
科学上网很多协议都是加密协议，在网络传输的过程中需要反复的加解密，性能不够的 CPU 加解密慢，从而导致上网体验不佳。

Q: 为什么不用软路由作为主路由，仅仅做一个旁路由？
A：1.我对软路由还并不是很了解。2.我并不认为软路由做主路由会很稳定，网路第一要义就是稳定，不稳定其他都是白瞎。3.并不是所有设备都需要科学上网，旁路由可以比主路由的方案更加方便设定。
