CentOS 7 安装 Portainer

当 Docker 镜像多了以后，使用终端来管理镜像就是一个不那么高效的做法了，我这里没有使用集群，所以挑选了一个简单的 Docker UI 来管理 Docker 镜像。

Portainer 是 Docker 的图形化管理工具，提供状态显示面板、应用模板快速部署、容器镜像网络数据卷的基本操作（包括上传下载镜像，创建容器等操作）、事件日志显示、容器控制台操作、Swarm集群和服务等集中管理和操作、登录用户管理和控制等功能 。可以说是简单但又很强大的一个图形化管理工具了。
