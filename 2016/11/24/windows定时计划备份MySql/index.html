使用 MySql 的 mysqldump 将数据库文件备份成 sql 文件。

Windows下备份

本地的数据库环境
MySql 安装环境：C:\MySql
数据库名称：bbs
root root
数据库备份目的地： D:/Data_BackUp

将上面的代码存储为 db_backup.bat

然后使用 Windows 自带的计划任务定时执行即可。
