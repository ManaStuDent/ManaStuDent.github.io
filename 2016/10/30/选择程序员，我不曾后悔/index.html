选择程序员，我不曾后悔

2016-10-30 阅读次数 0

文章目录

不知不觉工作已经有好几个月了，我的同学也大抵都开始工作了，按照年龄来算的话，应该是都已经毕业了。也不知道他们都选择了什么职业，或者是在那里工作。但我相信大家都是刚刚进入职场的新人，对自己选择的行业才刚刚有一个了解。当然也有进入职场了几年的老手，老司机。无论选择哪一行那一业，都有着自己说不出来的滋味。我想说，选择程序员，我不曾后悔。

一入博客深似海，从此节操是路人

写博客陆陆续续也有一年多了，我不知道当初是如何接触到博客的。可能是看到别人的博客而心生向往，然后就自己弄了一个。记得第一个博客是买了一个虚拟主机，然后用 WordPress 搭建起来的，也写了不少的东西，很多都是关于生活和自己的情感的。后来工作了，博客就改为用 GitHub Pages 搭建的静态页面，也多了很多关于技术的讨论和记录。但是和网上的大神们写的文章相比我这些只能算是，正的算是小打小闹。写的文章也根本不能叫做文章，只能够叫做随笔。对于这个博客我有一个目标，就是让越来越多的人知道我这个博客，然后让大家觉得我写的文章对他们有帮助，这样我的博客就算是达到目的了。但我也不能够忘记我写博客的最本质的目的————记录自己的成长。

Hello World

高考后我机缘巧合的选择了IT这一行业，那个时候对代码是一点都不懂。是一个彻彻底底的新人，麻瓜。但是我对编程却有一种莫名的好感。于是利用那年的暑假自学了 C#，这是我的启蒙语言。或许应该学习 C 才是，但是那个时候哪知道有什么语言。就知道我们以后专业课会开 C# 这一门课，于是乎跟着网上面的视频就学习了起来。学的还挺有感觉的。那个时候对编程是懵懂，是兴趣，是一种对未来的憧憬。

上学以后，是一段学习+放松的混合体。有发奋图强的奋斗 ，也有放纵的迷惘。但不论怎么样，这都过去了。

现在常常…想想和上学的时候差不多，工作+放松的混合体。需要改变自己的态度，不然时间一晃而过，留下的是换七八糟的记忆。那真的是一点都不愉快回忆。人既喜欢做舒服的事情，能躺着就不坐着，能坐着就不站着。但这还有什么意思呢？

选择程序员，我不曾后悔

我很庆幸自己能够选择程序员这一行，不仅仅是这一行能够给我带来不错的收入，更多的是因为我喜欢这一行的与时俱进，不断求新的感觉。当程序员不像是其他的传统行业，这一行需要不断的充实自己，不断的学习新知识，不断地解决新问题。我们和画家差不多，每天都去创造一些东西，去让这个世界更加的美好。但我现在还没有到这种等级，还没有熟练的掌握手中的画笔如何使用，色彩如何搭配，颜料怎样搭配。我需要的是时间的沉淀和自己的努力才可以让自己成为一名合格的画家。喜欢创造出一个网页的感觉，那个是自己的杰作，是自己的心血凝结出来的果实。要想创造出更加优美的果实，就需要向别人虚心学习，学习他们的技巧，手法和工具的应用。而且还需要自己去花时间琢磨，实践和研究。

杂七杂八的写了千把字，搞程序都是运用理性思维，而写文章运用的都是感性思维？不知道唉，反正我觉得现在写文章越来越没有逻辑可言了。这方面和我写代码一样，急需加强。
